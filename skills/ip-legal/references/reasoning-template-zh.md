# 知识产权中国法推理模板

> 用途：替代 U.S. Patent Act / Lanham Act / DMCA 等美国法推理

## 禁用的美国法推理模式

- ❌ Patent obviousness analysis (35 U.S.C. §103)
- ❌ First-to-invent vs First-to-file 混淆
- ❌ Fair use analysis (17 U.S.C. §107) — 美国四要素与中国合理使用不同
- ❌ Likelihood of confusion (Lanham Act §2(d)) — 中国商标第57条标准不同
- ❌ Trade dress 概念 — 中国反不正当竞争法第6条为商业混淆

## 中国法推理链

### 商标
```
Step 1 — 商标确权审查
  → 显著性？（商标法第9条）— 通用名称/描述性/缺乏显著特征不得注册
  → 绝对驳回理由？（商标法第10-12条）
  → 与他人在先申请商标近似？（商标法第30-31条）

Step 2 — 商标侵权分析
  → 双同侵权：相同商品+相同商标（商标法第57条(1)）
  → 似同侵权：类似商品+相同/近似商标+容易导致混淆（第57条(2)）
  → 认定"容易导致混淆"：商标近似程度+商品类似程度+相关公众注意力+实际混淆证据

Step 3 — 法律责任
  → 停止侵权 + 消除影响 + 赔偿损失
  → 赔偿计算顺序：权利人损失 → 侵权人获利 → 许可费倍数 → 法定赔偿（商标法第63条）
```

### 专利
```
Step 1 — 专利有效性分析
  → 新颖性：申请日以前没有同样的发明在国内/国外公开（专利法第22条）
  → 创造性：与现有技术相比具有突出的实质性特点和显著的进步

Step 2 — 侵权比对
  → 发明/实用新型：全部技术特征原则（全面覆盖原则）
  → 外观设计：整体视觉效果是否相同或近似

Step 3 — 赔偿
  → 实际损失→侵权获利→许可费合理倍数→法定赔偿
  → 惩罚性赔偿：故意侵权+情节严重 → 1-5倍（专利法第71条）
```

### 著作权
```
Step 1 — 作品认定（著作权法第3条）
  → 文学/艺术/科学领域内具有独创性+能以有形形式复制
Step 2 — 侵权比对："接触+实质性相似"
Step 3 — 合理使用判断（著作权法第24条，封闭清单，12项）
  → 注意：中国合理使用为封闭列举，与美国 open-ended fair use 不同
```
