# /ai-governance-legal:genai-compliance-check

1. AI服务类型识别：区分生成式AI服务、算法推荐服务、深度合成内容服务三类业态，确认是否具有舆论属性或社会动员能力。
2. 适配法规匹配：调取《生成式人工智能服务管理暂行办法》《算法推荐管理规定》《深度合成管理规定》对应条文。
3. 逐项合规清单核验：安全评估、算法备案、内容标识（显式/隐式水印）、训练数据合法性、用户权益保护五大维度。
4. 违规风险点位标记：梳理不合规项、法律后果、监管处罚标准。
5. 编制合规差距报告，输出《生成式AI产品境内上线合规检查报告》及整改行动清单。

## 详细说明

中国对人工智能采取了全球最敏捷、最具体的专项立法监管体系。AI系统在运行本技能时将深入穿透以下三大实务支柱：
- **生成式人工智能办法**：审查生成内容的真实性、准确性、客观性和公正性；训练数据来源的合法性（不得侵犯他人知识产权）；以及是否建立了防沉迷和不良信息过滤阻断机制。
- **深度合成规定**：针对技术中涉及人脸、人声等生物识别信息编辑、生成的，审查是否履行了用户知情同意，以及是否执行了不可逆、不可篡改的**显式与隐式内容标识（水印）**。
- **算法推荐规定**：针对具备舆论属性或社会动员能力的系统，审查是否提供用户一键关闭个性化推荐、选择或删除用户标签的功能，以及是否已履行**互联网信息服务算法备案**。

全品类AI适用科技伦理审查要求（《科技伦理审查办法（试行）》），杜绝违法不良内容生成传播。

### 输出格式示例

```markdown
# 生成式AI产品境内上线合规检查报告

## 一、 产品定性与监管法系匹配
- **产品名称**: [如：法务法理智能分析对话系统]
- **法系匹配**: 触发监管合规体系：
  1. 《生成式人工智能服务管理暂行办法》
  2. 《具有舆论属性或社会动员能力的互联网信息服务安全评估规定》

## 二、 专项合规差距分析矩阵（Gap Analysis）
| 法定合规指标 | 具体法规要求 | 系统当前现状评估 | 差距级别与具体整改动作 |
| :--- | :--- | :--- | :--- |
| **算法备案** | 算法上线30个工作日内通过网信办系统填报备案 | 尚未启动填报流程 | **高危阻断项**：必须在境内公开上线前完成备案并取得备案号 |
| **内容标识** | 对生成的图片/文本/语音进行显著与不显著标识 | 仅在页面右下角有文字声明 | **中危**：必须在后台输出流中嵌入隐式数字水印（Metadata级） |
| **训练数据合法性** | 确保不侵犯他人知识产权，具备合法语料授权 | 混合使用了开源数据集及未授权爬虫语料 | **高危**：清洗剔除高风险涉诉语料，补充签署授权书 |

## 三、 境内上线合规行动清单
1. 第一阶段：进行具有舆论属性服务的前置**安全评估**申报。
2. 第二阶段：通过国家互联网信息服务算法备案系统提交**算法机制机理**说明及承诺书。
```
