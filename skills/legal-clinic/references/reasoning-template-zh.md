# 法律援助/咨询中国法推理模板

> 用途：替代 U.S. pro se clinic / legal aid / hotline triage 等美国法推理

## 禁用的美国法推理模式

- ❌ U.S. legal aid eligibility (125% FPL) — 中国法律援助经济困难标准不同
- ❌ Malpractice insurance analysis — 中国律所执业风险体系不同
- ❌ Pro bono reporting (ABA Model Rule 6.1) — 中国律协公益法律服务指引不同
- ❌ Client trust accounting (IOLTA) — 中国律师收费管理办法框架

## 中国法推理链

### 初次咨询案件评估
```
Step 1 — 5W1H 事实提取
  → Who：各方当事人身份
  → What：核心诉求（法律上的可执行目标）
  → When：关键时间节点（合同签订日、侵权行为日、诉讼时效计算）
  → Where：管辖连接点
  → Why：法律争议焦点

Step 2 — 可诉性初步判断
  → 是否存在请求权基础？
  → 是否存在诉讼时效问题？
  → 是否存在仲裁/诉讼前置程序？（劳动仲裁、行政复议前置等）
  → 诉讼成本与预期收益的粗估（告得赢≠值得告）

Step 3 — 证据基础评估
  → 当事人手中现有哪些证据？
  → 哪些证据可以通过调查取证/法院调查令获取？
  → 证据缺失的风险有多大？（举证不能→败诉风险）
```

### 法律援助指引
```
Step 1 — 法律援助范围（法律援助法第25-31条）
  → 类型：刑事辩护全覆盖 + 六类民事/行政事项
  → 包括：请求国家赔偿/社会保险待遇/抚恤金/赡养费抚养费/劳动报酬/见义勇为
Step 2 — 经济困难标准
  → 各地有独立的认定标准（法律援助法第34条）
Step 3 — 值班律师制度
  → 认罪认罚案件中的值班律师提供法律帮助（刑事诉讼法第36条）
```

### 风险告知（必须）
```
- 诉讼时效是否即将届满？（立即行动窗口）
- 管辖是否有利？（异地诉讼成本）
- 对方是否有可执行财产？（赢了官司拿不到钱）
- 是否存在反诉风险？
- 预估时间周期：立案→开庭→一审判决→可能的二审/再审
```
