# /litigation-legal:pre-case-analysis

1. 判断案件所处阶段（起诉前/应诉/二审/再审），按对应侧重执行分析。
2. 按下方10步工作流推进，每一步分析到位后再进入下一步。
3. 严禁凭记忆引用法条——所有引用须标注来源并核实有效性。
4. 输出结构化 Markdown 报告。

# 委托代理前案件分析

## 适用范围

- **起诉前评估**（客户尚未起诉，评估能不能打、值不值得打）
- **应诉分析**（被告方，评估抗辩方向和反诉可能性）
- **二审上诉分析**（评估改判可能性和上诉策略）
- **再审分析**（评估再审事由和新证据效力）

## 工作纪律

### 法条引用
- 禁止凭记忆引用法条——必须通过在线检索验证，不得直接从训练数据调取
- 不确定的法条标注 [待人工核查]
- 引用法条时标注法律全称及制定/修订年份
- 已被修改或废止的法规不得引用

### 通用纪律
- 不虚构案例、不编造法条、不遗漏风险（即使对委托人不利）
- 不擅自定论——争议问题提供多种分析路径
- 信息不完整时标注 [假设：……]

## 10步工作流程

### 第一步：案件阶段判断与材料分类

先判断案件所处阶段（起诉前/应诉/二审/再审），不同阶段材料类型和分析侧重不同：

| 阶段 | 典型材料 | 分析侧重 |
|------|----------|----------|
| **起诉前（原告方）** | 合同、函件、聊天记录、转账凭证、发票、损失证据 | 请求权基础、证据充分性、诉讼方案 |
| **应诉（被告方）** | 起诉状+合同、己方证据 | 抗辩方向、反诉可能性、风险敞口 |
| **二审** | 一审判决书+原始材料 | 一审裁判错误识别、改判可能性 |
| **再审** | 一二审判决书+新证据 | 再审事由、新证据效力 |

### 第二步：当事人与利益图谱

- 各方当事人全称、主体资格（法人/自然人/个体工商户）
- 委托人在案件中的角色和立场
- 是否存在需要追加的第三人（共同被告/担保人/实际责任方）
- 各方利益诉求及潜在冲突点

### 第三步：事实梳理与法律关系识别

- 案件核心事实按时间线梳理
- 识别全部法律关系（不仅看委托人和直接对手方，还包括历史遗留关系、行政关系、第三方权利冲突）
- 明确请求权基础/抗辩事由的法律依据
- 标注事实缺口——哪些关键事实不清楚，需要追问或补充材料

### 第四步：证据评估

按证据三性（真实性/合法性/关联性）评估已有证据：
- 已有证据清单（标注证明力：强/中/弱）
- 缺失的必需证据（标注致命/重要/辅助）
- 补强路径建议

### 第五步：法律法规检索

按层级检索：
1. 法律（全国人大及其常委会）——核实是否为现行有效最新版本
2. 行政法规（国务院）
3. 司法解释（最高法/最高检）
4. 部门规章（各部委）
5. 地方性法规/规章（如涉及）
6. 指导性案例/典型案例

### 第六步：风险识别与分层

按四级分层标注风险：

| 级别 | 说明 | 示例 |
|------|------|------|
| 🔴**结构性风险** | 影响案件根基的问题 | 权利来源缺陷、法律禁止性规定、诉讼时效已过 |
| 🟠**重大风险** | 不处理会导致严重后果 | 证据链断裂、超高违约金、管辖不利 |
| 🟡**一般风险** | 常规需关注的问题 | 部分证据证明力不足 |
| 🟢**潜在风险** | 需预判防范的问题 | 政策变化、对方可能的反诉 |

### 第七步：诉讼策略预判

- 我方最优诉讼路径（请求权/抗辩路径）
- 对方可能的反驳路径及应对
- 调解/和解空间评估
- [有裁判文书时] 一审裁判错误定位（事实认定/法律适用/程序问题）

### 第八步：量化评估

- 诉讼请求金额/抗辩金额的量化计算
- 诉讼费、保全费、鉴定费等成本预估
- 时间周期预估（一审约3-6个月，二审约3个月）
- 改判/胜诉可能性综合分析（客观评估，不承诺结果）

### 第九步：反诉/交叉请求评估

- 是否存在可向对方主张的独立请求
- 反诉的法定条件是否具备（本诉的牵连性、法庭辩论终结前）
- 反诉策略建议

### 第十步：输出分析报告

`
