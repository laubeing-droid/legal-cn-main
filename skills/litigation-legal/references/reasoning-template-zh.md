# 诉讼/仲裁中国法推理模板

> 用途：替代 adversarial / discovery-driven 推理骨架，按中国民商事审判逻辑进行分析

## 禁用的美国法推理模式

- ❌ Discovery planning — 中国无美式证据开示
- ❌ Deposition strategy — 中国无庭前宣誓证言制度
- ❌ Summary judgment motion — 中国无即决判决
- ❌ Privilege log — 中国无 attorney-client privilege 制度
- ❌ Hearsay analysis — 中国证据法不使用传闻证据规则

## 中国法推理链

### 诉讼策略分析
```
Step 1 — 请求权基础分析
  → 可能的请求权基础有哪些？（逐一列举法条依据）
  → 各请求权的构成要件分别是什么？
  → 本案事实能满足哪些构成要件？

Step 2 — 程序前提审查
  → 管辖：级别管辖（标的额）+ 地域管辖（被告住所地/合同履行地/侵权行为地）
  → 诉讼时效：是否届满？是否存在中断/中止事由？（民法典第188-199条）
  → 主体适格：原告/被告是否为适格当事人？

Step 3 — 证据评估（中国证据法框架）
  → 证据三性：真实性/合法性/关联性
  → 举证责任分配：一般规则（谁主张谁举证）+ 举证责任倒置情形
  → 证明标准：民事诉讼一般标准（高度盖然性）+ 特殊标准（排除合理怀疑）

Step 4 — 类案预测
  → 检索该法院/该法官同类案件裁判倾向
  → 类案裁判规则：同案是否同判？
```

### 文书推理
```
起诉状 → 诉讼请求明确 + 事实理由支撑 + 法条依据一一对应
代理词 → 围绕争议焦点 + 构成要件分析 + 法条+类案双重支撑
证据目录 → 按三性分组 + 证明目的清晰 + 原件/复印件标注
答辩状 → 逐项回应 + 程序抗辩先于实体抗辩
```
