# 产品合规中国法推理模板

> 用途：替代 U.S. product liability / FDA/CPSC regulation / FTC advertising rules 等美国法推理

## 禁用的美国法推理模式

- ❌ Strict product liability (Restatement 2d §402A) — 中国为《民法典》第1202-1207条，责任体系与认定标准不同
- ❌ FDA premarket approval — 中国有独立的药品/医疗器械注册监管体系
- ❌ FTC unfair/deceptive practices (15 U.S.C. §45) — 中国为《广告法》+《反不正当竞争法》框架
- ❌ Class action litigation — 中国为代表人诉讼（民事诉讼法第56-57条），实践差异显著

## 中国法推理链

### 产品上市合规
```
Step 1 — 产品定性
  → 普通消费品？食品/药品/医疗器械/化妆品？婴幼儿产品？
  → 各品类纳入不同监管体系（食药监局/市场监管局/卫健委）

Step 2 — 产品质量标准
  → 国标（GB）、行标、企标
  → 是否为强制性标准？（标准化法第2条）

Step 3 — 标识要求
  → 中文标识（产品质量法第27条）
  → 生产日期/安全使用期/失效日期
  → 警示标志/中文警示说明（危险品/剧毒品/放射性物品）
```

### 广告合规
```
Step 1 — 广告内容红线（广告法第9条）
  → "最"级用语禁止（国家级、最高级、最佳）
  → 损害国家尊严或利益的内容
  → 妨碍社会公共秩序或违背社会良好风尚

Step 2 — 特殊品类广告限制
  → 医疗/药品/保健食品/农药/兽药：须经审查（广告法第46条）
  → 烟草/处方药：禁止发布广告（第15/22条）
  → 婴儿乳制品/饮料/其他食品：不得声称可替代母乳（第20条）

Step 3 — 比较广告
  → 禁止贬低其他生产经营者的商品或服务（广告法第13条）
  → 对比须基于事实且不会引人误解
```

### 产品责任
```
Step 1 — 责任主体识别（民法典第1202-1207条）
  → 生产者：严格责任（不问过错，但可主张法定免责事由）
  → 销售者：过错责任（不能指明缺陷生产者/供货者 → 自己承担）

Step 2 — 缺陷认定
  → 设计缺陷/制造缺陷/警示说明缺陷
  → 不合理危险的判断：消费者期待标准 + 风险-效用分析

Step 3 — 损害赔偿范围
  → 人身损害赔偿 + 财产损失
  → 惩罚性赔偿（民法典第1207条：明知缺陷 + 造成死亡/健康严重损害）
```

### 电商合规
```
Step 1 — 平台责任（电子商务法第38条）
  → 平台对平台内经营者的资质审核义务
  → 平台对消费者安全保障义务（关系消费者生命健康的商品/服务）
Step 2 — 算法合规
  → 个性化推荐须提供非个性化选项
  → 《网络交易监督管理办法》信息公示义务
```
