# 法学教育中国法推理模板

> 用途：替代 U.S. law school pedagogy / case method / Socratic method 等美国法推理

## 禁用的美国法推理模式

- ❌ Case method — 中国法学教育以法条解释为主，判例为辅
- ❌ IRAC (Issue-Rule-Analysis-Conclusion) — 中式法律文书使用三段论（法律规定-案件事实-裁判结论）
- ❌ Bluebook citation — 中国引用规范不同
- ❌ Law review note structure — 中国法学论文为"问题提出-学理分析-制度建议"结构

## 中国法推理链

### 案例分析三段论
```
Step 1 — 法律规定（大前提）
  → 找法：适用的法律规范是什么？法条的构成要件是什么？
  → 注意法律位阶：宪法 > 法律 > 行政法规 > 地方性法规/规章
  → 注意法条竞合：一般法与特别法的适用关系（如民法典总则编 vs 合同编）

Step 2 — 案件事实（小前提）
  → 固定事实：哪些是证据能认定的客观事实？
  → 事实归类：这些事实对应法条的哪个构成要件要素？
  → 证据缺口：哪些要件缺乏证据支撑？

Step 3 — 裁判结论（推理结果）
  → 事实能否涵射到法律规范之下？
  → 法律效果是什么？
```

### 民法请求权基础分析法
```
Step 1 — 确定请求权目标（当事人想要什么法律效果？）
Step 2 — 寻找请求权基础（合同请求权/侵权请求权/物权请求权/不当得利/无因管理）
Step 3 — 检视构成要件要素是否满足
Step 4 — 排除对立规范（是否存在权利障碍/权利消灭/权利抗辩事由？）
Step 5 — 确定法律效果
```

### 论文写作结构
```
中国法学论文标准结构：
1. 问题提出（立法/司法/理论争议）
2. 域外经验与理论资源（比较法分析，注意法系差异）
3. 规范分析（现行法条+司法解释的体系化解读）
4. 实证/案例佐证（判决文书数据 + 典型案例）
5. 法律完善建议（立法论/解释论二元区分）
```

## 来源标签
- `[民法典第X条]` — 引用法律原文
- `[指导案例X号]` — 引用最高法指导性案例
