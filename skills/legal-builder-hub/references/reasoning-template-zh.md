# 技能构建器中国法推理模板

> 用途：确保新创建的法律子技能适配中国法律环境

## 禁令

- ❌ 禁止直接套用美国法工作流结构
- ❌ 禁止引入中国法不存在的制度作为推理依据
- ❌ 禁止创建"中美法律对比"类技能（本仓库定位为中国法优先）

## 中国法子技能设计规范

### 创建新技能时必须回答
```
1. 这个技能解决什么中国法律问题？
2. 它的上游来源是什么？（anthropic原始 / zhou210712中文化 / 本仓库新增）
3. 它使用了哪些中国法概念？（须在 22-blocked-concepts.md 中不存在于阻断清单）
4. 它的推理链是否符合中国法逻辑？（不得残留美国法推理模式）
```

### 技能结构规范
```
skill-name/
├── SKILL.md           # 技能定义（中文，含推理链说明）
├── references/        # 引用文件
│   └── （仅引用中国法来源和本仓库对齐框架）
```

### 推理链注入检查清单
- [ ] 技能中是否包含"先确定请求权基础"步骤？（中国民商事诉讼核心）
- [ ] 技能中是否默认使用"三段论"推理而非IRAC？
- [ ] 引用法条是否标注具体条目？（民法典第X条第Y款）
- [ ] 是否避免了阻断清单中的29个美国法概念？
- [ ] 是否标注了中国特色制度？（★ 中国特色制度标签）
- [ ] 是否包含强制免责声明？
