# 合同审查清单 — AI 可执行版

> 基于 contract-review-playbook.md，逐条可勾选。AI 审查合同时按本清单逐项执行。
> 来源标签：`[民法典第X条]` | `[合同监管办法第X条]` | `[消保条例第X条]`

## 第一阶段：合同识别（3 项）

- [ ] 1.1 识别合同类型：有名/无名/混合？（民法典合同编第二分编）
- [ ] 1.2 识别合同性质：双务/单务？有偿/无偿？诺成/实践？
- [ ] 1.3 识别审查立场：甲方/乙方/中性？→ 影响后续修改建议方向

## 第二阶段：效力审查（12 项）

### 成立要件
- [ ] 2.1 要约+承诺是否完成？（民法典第472-477条）
- [ ] 2.2 合同形式是否满足法定要求？（书面/口头/电子）
- [ ] 2.3 当事人是否具备民事行为能力？（民法典第143条）

### 效力状态
- [ ] 2.4 是否存在违反强制性规定情形？（民法典第153条第1款）→ 无效
- [ ] 2.5 是否违背公序良俗？（民法典第153条第2款）→ 无效
- [ ] 2.6 是否存在恶意串通损害他人权益？（民法典第154条）→ 无效
- [ ] 2.7 是否存在重大误解？（民法典第147条）→ 可撤销
- [ ] 2.8 是否存在欺诈？（民法典第148条）→ 可撤销
- [ ] 2.9 是否存在胁迫？（民法典第150条）→ 可撤销
- [ ] 2.10 是否显失公平？（民法典第151条）→ 可撤销
- [ ] 2.11 是否存在无权代理？（民法典第171条）→ 效力待定
- [ ] 2.12 是否存在无权处分？（民法典第597条）→ 效力待定

## 第三阶段：格式条款专项（8 项）

> 适用场景：合同一方为经营者，使用预先拟定、重复使用的条款

- [ ] 3.1 是否以显著方式提请对方注意重大利害关系内容？（合同监管办法第6条）
- [ ] 3.2 是否按对方要求对格式条款予以说明？（民法典第496条）
- [ ] 3.3 是否存在免除己方责任、加重对方责任的内容？（民法典第497条）→ 无效
- [ ] 3.4 是否存在排除对方主要权利的内容？（民法典第497条）→ 无效
- [ ] 3.5 是否存在"最终解释权"条款？（合同监管办法第8条）→ 无效
- [ ] 3.6 是否借助技术手段强制交易？（合同监管办法第9条）→ 违法
- [ ] 3.7 格式条款与非格式条款不一致时，是否优先适用非格式条款？（民法典第498条）
- [ ] 3.8 格式条款有两种以上解释的，是否采用对提供方不利的解释？（民法典第498条）

## 第四阶段：核心条款审查（15 项）

### 必备条款（民法典第470条）
- [ ] 4.1 标的：是否明确？（数量、质量标准、规格型号）
- [ ] 4.2 价款/报酬：是否明确？（金额、币种、计价方式、支付时间）
- [ ] 4.3 履行期限：是否明确？（起止时间、阶段里程碑）
- [ ] 4.4 履行地点和方式：是否明确？
- [ ] 4.5 违约责任：是否具体可执行？（非笼统"承担违约责任"）
- [ ] 4.6 争议解决方法：是否明确？（诉讼/仲裁，仲裁委名称须精确）

### 违约责任
- [ ] 4.7 违约金是否过高？（民法典第585条：超过损失30%可请求减少）
- [ ] 4.8 定金是否超过合同标的额20%？（民法典第586条）
- [ ] 4.9 免责条款是否排除人身伤害责任？（民法典第506条）→ 无效
- [ ] 4.10 免责条款是否排除故意或重大过失责任？（民法典第506条）→ 无效

### 其他关键条款
- [ ] 4.11 管辖法院/仲裁机构是否对己方有利？
- [ ] 4.12 是否存在单方任意解除权？（过高风险，建议限制）
- [ ] 4.13 保密条款范围是否合理？（过宽或过窄）
- [ ] 4.14 知识产权归属是否明确？（委托开发/合作开发场景尤其重要）
- [ ] 4.15 不可抗力条款是否包含"政府行为"？（中国实务建议包含）

## 第五阶段：消费者合同专项（6 项）

> 适用场景：合同相对方为消费者

- [ ] 5.1 是否存在"自动续费"？是否在到期前显著提示？（消保条例第10条）
- [ ] 5.2 搭售/组合销售是否逐一显著提示？（消保条例第11条）
- [ ] 5.3 是否利用技术手段实施差别待遇（"大数据杀熟"）？（消保条例第9条）
- [ ] 5.4 预付式消费是否约定了退款机制？（消保条例第19条）
- [ ] 5.5 退货/换货/修理的期限和方式是否合法？（消费者权益保护法第24条）
- [ ] 5.6 是否有限制消费者投诉/诉讼权利的条款？→ 无效

## 第六阶段：数据条款专项（4 项）

- [ ] 6.1 是否涉及个人信息处理？→ 需独立授权+隐私政策链接
- [ ] 6.2 是否涉及敏感个人信息？→ 需单独同意+必要性说明
- [ ] 6.3 是否涉及数据出境？→ 需安全评估/SCC/认证（三选一）
- [ ] 6.4 数据处理方（受托方）的义务是否明确？（个保法第21条）

## 第七阶段：风险评级与输出（3 项）

- [ ] 7.1 逐条标注风险等级：🔴高 🟡中 🟢低
- [ ] 7.2 每条风险附法条依据 + 修改建议（推荐方案 + 最低可接受方案）
- [ ] 7.3 输出总体风险评级 + 谈判策略建议

---

> **总计：51 项检查点** | 覆盖民法典合同编 + 消保法 + 合同监管办法 + 个保法
