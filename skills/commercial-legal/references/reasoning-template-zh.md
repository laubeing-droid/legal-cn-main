# 商事合同中国法推理模板

> 用途：替代 common-law / adversarial 推理骨架，按中国民商事审判逻辑进行分析

## 禁用的美国法推理模式

- ❌ IRAC (Issue-Rule-Analysis-Conclusion) — 美式备忘录结构
- ❌ Issue spotting — 判例法下的争点识别
- ❌ Consideration analysis — 对价分析
- ❌ Parol evidence rule — 口头证据规则
- ❌ Reasonable person standard — 美国侵权法合理人标准

## 中国法推理链

### 合同审查
```
Step 1 — 合同性质认定
  → 有名合同 / 无名合同（民法典第467条）
  → 单务 / 双务（民法典第469-470条）

Step 2 — 成立与效力审查
  → 要约+承诺完成？要约撤回/撤销条件满足？（民法典第472-477条）
  → 格式条款有效？（民法典第496-498条）
  → 是否存在民法典第153条无效情形？（违反强制性规定/违背公序良俗）

Step 3 — 核心条款检视
  → 数量/质量/价款/履行期限方式/违约责任/争议解决
  → 违约金是否过高？（民法典第585条 → 违约金超过损失30%可申请减少）

Step 4 — 风险条款识别
  → 免责条款有效性（民法典第506条）
  → 不可抗力/情势变更（民法典第180条/第533条）
  → 格式条款提示说明义务履行（民法典第496条）

Step 5 — 争议解决条款
  → 管辖约定是否有效？（民事诉讼法第34条）
  → 仲裁条款是否明确？（仲裁法第16条）
```

### 合同纠纷代理
```
Step 1 — 请求权基础确定
  → 主张何方构成违约？请求权依据？（民法典第577/584条）
  → 是否存在同时履行抗辩/先履行抗辩/不安抗辩？（民法典第525-527条）

Step 2 — 构成要件检视
  → 违约行为是否成立？
  → 损失与违约行为因果关系？
  → 是否存在免责事由？（不可抗力/债权人过错）

Step 3 — 责任范围计算
  → 实际损失 + 可得利益（民法典第584条）
  → 是否有违约金约定？（民法典第585条调整）
  → 是否有减损义务？（民法典第591条）
```
