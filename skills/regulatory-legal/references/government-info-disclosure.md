# 政府信息公开条例实务要点（2019修订版）

> 来源：培训课件《新〈政府信息公开条例〉解读》（已脱敏）

---

## 一、修订背景
1. 贯彻落实党中央国务院关于政府信息公开工作的要求
2. 现行政府信息公开的广度和深度已不能满足人民群众日益增长的需要
3. 政府信息公开出现新问题、新情况、新经验，迫切需要总结规范提升

---

## 二、总则修订要点

### 1. 重新明确政府信息概念
政府信息，是指行政机关在履行行政管理职能过程中制作或者获取的，以一定形式记录、保存的信息。

> 需加工、分析的信息 → 行政机关可以不予提供（第37条区分处理除外）

### 2. 信息公开领导机制
- 国务院办公厅 → 全国政府信息公开工作主管部门
- 县级以上地方人民政府办公厅（室） → 本行政区域主管部门
- 实行垂直领导的部门的办公厅（室） → 本系统主管部门

### 3. 基本原则
- **以公开为常态、不公开为例外**
- 公正、公平、合法、便民

### 4. 申请登记制度
行政机关应当建立健全政府信息公开申请登记、审核、办理、答复、归档的工作制度。

---

## 三、公开主体和范围

### 公开主体确定规则
| 情形 | 公开主体 |
|:-----|:---------|
| 行政机关制作的政府信息 | 制作该信息的行政机关 |
| 从公民/法人/其他组织获取的政府信息 | 保存该信息的行政机关 |
| 获取的其他行政机关的政府信息 | 制作或最初获取该信息的行政机关 |
| 派出机构/内设机构以自己名义履职 | 该派出机构/内设机构 |

### 不予公开范围
- 国家秘密（绝对不公开）
- 法律、行政法规禁止公开
- 公开后可能危及"三安全一稳定"（国家安全/公共安全/经济安全/社会稳定）
- 商业秘密/个人隐私（经权利人同意或不公开可能对公共利益造成重大影响的可以公开）
- 内部事务信息（人事管理/后勤管理/内部工作流程）
- 过程性信息（讨论记录/过程稿/磋商信函/请示报告）
- 行政执法案卷信息

---

## 四、主动公开

### 主动公开范围（第20条）
- 行政法规、规章和规范性文件
- 机关职能、机构设置、办公地址、联系方式
- 国民经济和社会发展规划、专项规划、区域规划
- 财政预算、决算信息
- 行政事业性收费项目及依据、标准
- 政府集中采购项目
- 行政许可/行政处罚/行政强制
- 扶贫、教育、医疗、社保、就业
- 突发公共事件的应急预案、预警信息及应对情况
- 环境保护、公共卫生、安全生产、食药、产品质量

---

## 五、依申请公开

### 申请形式
- 书面（含数据电文形式）
- 口头（行政机关可代填）

### 答复时限
- 一般：收到申请之日起20个工作日内
- 延期：经负责人同意可延长20个工作日
- 征求第三方意见时间不计算在内

### 答复方式
| 情形 | 答复 |
|:-----|:-----|
| 已主动公开 | 告知获取方式和途径 |
| 可以公开 | 提供政府信息 |
| 不予公开 | 说明理由 |
| 不属于本机关公开 | 告知并说明理由；能确定公开机关的告知该机关 |
| 信息不存在 | 告知 |
| 申请内容不明确 | 告知更改、补充 |

### 救济途径
- 举报：向上级行政机关/政府信息公开工作主管部门
- 行政复议：收到答复之日起60日内
- 行政诉讼：收到答复之日起6个月内

## AI 处理政府信息公开案件的核查清单

| # | 核查项 |
|:--|:-----|
| 1 | 申请信息是否属于"政府信息"（履职过程中制作/获取+记录/保存）？ |
| 2 | 申请是否需加工/分析？（行政机关可不予提供） |
| 3 | 是否属于不予公开范围（国家秘密/内部事务/过程性/执法案卷）？ |
| 4 | 是否涉及商业秘密/个人隐私？（需征求第三方意见） |
| 5 | 被告是否适格的公开主体？ |
| 6 | 答复是否在20个工作日内？是否超期？ |
| 7 | 是否告知救济途径和期限？ |
