# 行政/监管中国法推理模板

> 用途：替代 APA / Chevron deference / rulemaking notice-and-comment 等美国法推理

## 禁用的美国法推理模式

- ❌ Chevron deference — 中国行政诉讼中法院对行政行为进行独立司法审查，不采纳行政机关法律解释优先原则（《行政诉讼法》第6条）
- ❌ APA rulemaking procedure — 中国有独立的行政法规/规章制定程序
- ❌ Takings clause analysis — 中国征收征用制度为《民法典》第243条+《土地管理法》框架
- ❌ Exhaustion of administrative remedies — 中国行政复议非诉讼前置（复议前置情形有限）

## 中国法推理链

### 行政行为合法性审查
```
Step 1 — 行政行为类型识别
  → 行政许可？行政强制？行政处罚？行政协议？
  → 行政规范性文件是否一并审查？（行政诉讼法第53条）

Step 2 — 职权依据
  → 行政机关是否有法定职权？（超越职权→可撤销/无效）
  → 是否滥用职权？

Step 3 — 事实认定与证据
  → 主要证据是否充分？
  → 适用法律是否正确？

Step 4 — 程序合法性
  → 是否履行法定程序？（告知/听证/说明理由/送达）
  → 重大程序违法→可撤销

Step 5 — 裁量合理性
  → 行政处罚是否显失公平？
  → 行政诉讼法第70条：明显不当→可撤销
```

### 企业行政合规
```
Step 1 — 监管领域识别
  → 市场监管/网信/金融/药监/环保，多部门可能交叉监管

Step 2 — 行政许可清单
  → 市场准入负面清单制度
  → 是否有未获得的许可证/批准/备案？

Step 3 — 日常合规义务
  → 年度报告/专项报告/备案义务
  → 信息公示义务
  → 配合检查调查义务
```

### 政企互动流程
```
投诉举报 → 行政调查 → 约谈 → 听证 → 处罚决定 → 行政复议/诉讼
```
注意：约谈是中国特有制度，不等于处罚决定，但具有合规信号意义。
