# /employment-legal:labor-dispute-analysis

1. 分析涉案事实是否符合劳社部发〔2005〕12号文的"劳动关系三要素"，确认劳动关系性质。
2. 审查用人单位解除或终止劳动合同的实体与程序合法性（区分协商解除、过错辞退、无过错辞退、违法辞退四类）。
3. 准确适用中国用工标准，测算经济补偿金（N）、违法解除赔偿金（2N）或代通知金（N+1）及各项欠付薪资。
4. 核查仲裁时效（1年）与双方证据效力，识别举证责任分配。
5. 审议工伤情形下的伤残等级对应待遇、停工留薪期等配套责任。
6. 生成《劳动争议仲裁/诉讼策略应对报告》。

## 详细说明

本技能专注于中国境内的劳动纠纷实务路径：
- **劳动关系认定**：严格对照劳社部发〔2005〕12号，从主体资格、用工管理（从属关系）和业务组成三方面区分劳动关系与劳务/合作关系。
- **解雇合法性**：对照《中华人民共和国劳动合同法》第39条（过错性解除）、第40条（非过错性解除）及第46条，审查企业解除程序（对照第48条违法解除认定标准）（如是否通知工会、是否有送达回执）。
- **金额精准测算**：结合当地社平工资上限（3倍封顶）和最低工资标准，执行中国法特有的 N / 2N / N+1 测算。
