# 劳动法中国法推理模板

> 用途：替代 at-will employment / severance negotiation 等美国法推理骨架

## 禁用的美国法推理模式

- ❌ At-will employment — 中国无法定任意解雇制
- ❌ Severance negotiation — 中国为法定经济补偿/赔偿
- ❌ Wrongful termination framing — 中国分"违法解除"与"经济性裁员"
- ❌ Employment handbook as contract — 中国以法定权利义务为主

## 中国法推理链

### 解除劳动合同分析
```
Step 1 — 解除类型识别
  → 协商一致解除？（劳动合同法第36条）
  → 劳动者单方解除？（第37条：提前30日通知 / 第38条：用人单位过错）
  → 用人单位单方解除？（第39条：过失性解除 / 第40条：无过失性解除）

Step 2 — 经济性裁员专项
  → 是否满足第41条门槛：裁减人员>=20人或占职工总数10%以上？
  → 程序要件：提前30日向工会/全体职工说明 + 听取意见 + 向劳动行政部门报告
  → 优先留用范围：无固定期限合同/长期合同/家庭无其他就业人员
  → 禁止裁减：孕期/产期/哺乳期女职工（第42条）

Step 3 — 补偿/赔偿计算
  → 经济补偿：N = 工作年限 × 月工资（第47条）
  → 违法解除赔偿金：2N（第87条）
  → 三种情形必须给N：劳动者被迫解除（第38条）、用人单位提出协商解除（第36条）、
    无过失性解除（第40条）、经济性裁员（第41条）、合同期满终止（第46条）
```

### 劳动争议路径
```
Step 1 — 必经仲裁前置程序
  → 劳动仲裁（劳动争议调解仲裁法第5条）
  → 仲裁时效：1年（从知道权利被侵害之日起）
Step 2 — 对仲裁不服可起诉
  → 15日内向有管辖权法院提起诉讼
```
