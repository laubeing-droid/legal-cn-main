## 第一阶段：招聘与入职（10 项）

### 招聘环节
- [ ] 1.1 招聘广告是否含有歧视性内容？（性别/民族/宗教/残疾/户籍）→ 违法（就业促进法）
- [ ] 1.2 是否扣押劳动者证件或收取财物？（劳动合同法第9条）→ 违法，处500-2000元罚款/人
- [ ] 1.3 是否建立职工名册？（劳动合同法第7条）→ 用工之日起建立

### 合同签订
- [ ] 1.4 是否自用工之日起1个月内签订书面劳动合同？（劳动合同法第10条）
- [ ] 1.5 未签合同超过1个月？→ 支付2倍工资（劳动合同法第82条）
- [ ] 1.6 未签合同超过1年？→ 视为无固定期限合同（劳动合同法第14条）
- [ ] 1.7 试用期长度是否合法？（合同期3月-1年→1月；1-3年→2月；3年以上→6月。劳动合同法第19条）
- [ ] 1.8 试用期工资是否≥同岗位最低档80%且≥当地最低工资？（劳动合同法第20条）
- [ ] 1.9 是否签署竞业限制协议？（仅限高管/高级技术人员/其他保密人员，劳动合同法第24条）
- [ ] 1.10 竞业限制范围是否合理？（期限≤2年，需支付经济补偿，劳动合同法第23-24条）

## 第二阶段：在职管理（12 项）

### 工时与休假
- [ ] 2.1 工时制度：标准/综合计算/不定时？是否经劳动部门审批？
- [ ] 2.2 加班是否支付加班费？（工作日150%/休息日200%/法定假日300%，劳动法第44条）
- [ ] 2.3 是否存在"996"或变相强制加班？→ 违法（人社部+最高法典型案例）
- [ ] 2.4 年休假是否依法安排？（累计工作1-10年→5天；10-20年→10天；20年以上→15天）
- [ ] 2.5 产假/陪产假是否合规？（女职工劳动保护特别规定第7条：98天+地方奖励假）

### 薪酬与社保
- [ ] 2.6 工资是否≥当地最低工资标准？（劳动合同法第20条）
- [ ] 2.7 是否依法缴纳社会保险（五险）？（社会保险法）
- [ ] 2.8 是否存在"社保折现"替代缴纳？→ 违法，协议无效
- [ ] 2.9 工资支付周期是否≤1个月？（工资支付暂行规定第7条）

### 规章制度
- [ ] 2.10 规章制度是否经民主程序制定？（职工代表大会讨论+协商，劳动合同法第4条）
- [ ] 2.11 规章制度是否向劳动者公示或告知？（未公示不能作为依据）
- [ ] 2.12 规章制度内容是否合法？（末位淘汰/罚款/体罚→违法。指导案例18号：末位淘汰≠不胜任工作）

## 第三阶段：劳动关系变更（5 项）

- [ ] 3.1 调岗/调薪是否经双方协商一致？（劳动合同法第35条）→ 单方调岗需审查合理性
- [ ] 3.2 是否涉及医疗期满调岗？（劳动合同法第40条第1项）
- [ ] 3.3 是否涉及不胜任工作调岗？（需培训或调岗，仍不胜任才可解除。劳动合同法第40条第2项）
- [ ] 3.4 是否涉及"三期"女职工调岗？（女职工劳动保护特别规定第6条：减轻劳动量或调岗）
- [ ] 3.5 变更后是否签订书面变更协议？（建议形式：补充协议/变更确认书）

## 第四阶段：解除与终止（14 项）

### 协商解除
- [ ] 4.1 是否双方协商一致？（劳动合同法第36条）→ 用人单位提出需支付经济补偿

### 劳动者单方解除
- [ ] 4.2 劳动者提前30日书面通知？（劳动合同法第37条）→ 无需补偿
- [ ] 4.3 劳动者以未足额支付工资/未缴社保为由解除？（劳动合同法第38条）→ 用人单位需支付补偿

### 用人单位单方解除
- [ ] 4.4 过失性解除（即时解除）：是否满足法定情形？（严重违纪/严重失职/追究刑事责任等。第39条）
- [ ] 4.5 过失性解除：违纪事实是否有证据？规章制度是否有规定？是否通知工会？（第43条）
- [ ] 4.6 无过失解除：是否属于第40条情形？（医疗期满/不胜任/客观情况重大变化）
- [ ] 4.7 无过失解除：是否提前30日通知或支付代通知金？（N+1）
- [ ] 4.8 是否属于禁止解除情形？（第42条：职业病/工伤/医疗期/"三期"女工/连续工作15年+距退休不足5年）

### 经济性裁员
- [ ] 4.9 是否满足裁员条件？（第41条：重整/经营困难/转产/重大技术革新/其他客观经济情况）
- [ ] 4.10 裁员超过20人或占职工总数10%以上？（需提前30日向工会或全体职工说明+报告劳动部门）
- [ ] 4.11 优先留用人员是否正确？（第41条第2款：较长合同期限/无固定期限/家庭无其他就业人员）

### 补偿与赔偿
- [ ] 4.12 经济补偿（N）是否按法定标准？（每满1年=1个月工资；6月-1年=1个月；不满6月=0.5个月。第47条）
- [ ] 4.13 违法解除是否支付赔偿金（2N）？（第87条）
- [ ] 4.14 是否出具解除/终止证明并在15日内办理档案和社保转移？（第50条）

## 第五阶段：争议应对（5 项）

- [ ] 5.1 是否收到劳动仲裁申请书？→ 15日内提交答辩书（劳动争议调解仲裁法第30条）
- [ ] 5.2 是否属于用人单位举证责任倒置情形？（工资支付/考勤/解除决定。劳动争议调解仲裁法第6条）
- [ ] 5.3 仲裁时效是否届满？（1年，自知道权利被侵害之日起。第27条）
- [ ] 5.4 是否涉及工伤认定？（单位应在30日内申请，否则职工可在1年内自行申请）
- [ ] 5.5 是否需要申请财产保全？（防止转移财产，劳动争议调解仲裁法第44条）
