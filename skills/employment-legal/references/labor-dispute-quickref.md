# 中国劳动争议裁决实务速查手册

## 1. 劳动关系认定"三要素"（劳社部发〔2005〕12号）
同时具备下列情形的，劳动关系成立：
1. 用人单位和劳动者符合法律、法规规定的主体资格；
2. 用人单位依法制定的各项劳动规章制度适用于劳动者，劳动者受用人单位的劳动管理，从事用人单位安排的有报酬的劳动；
3. 劳动者提供的劳动是用人单位业务的组成部分。

## 2. 解除补偿与赔偿测算基准（《劳动合同法》第47条、第48条、第87条）
- **经济补偿金（N）**：按劳动者在本单位工作年限，每满一年支付一个月工资。六个月以上不满一年的，按一年计算；不满六个月的，向劳动者支付半个月工资的经济补偿。
  - *注：工作均值工资高于当地社平工资3倍的，按3倍封顶，年限最高不超过12年。*
- **代通知金（N+1）**：仅适用于《劳动合同法》第40条三种情形（医疗期满、不胜任工作、客观情况重大变化），未提前30天书面通知的，额外支付一个月工资。
- **违法解除赔偿金（2N）**：依据《中华人民共和国劳动合同法》第48条及第87条，用人单位违反本法规定解除或者终止劳动合同的，应当依照本法第四十七条规定的经济补偿标准的**二倍**向劳动者支付赔偿金。

## 3. 工伤一次性伤残补助金速算表（《工伤保险条例》第35-37条）
依据伤残等级，由工伤保险基金支付的一次性伤残补助金标准（按本人工资计）：
- 一级伤残 = 27个月
- 二级伤残 = 25个月
- 三级伤残 = 23个月
- 四级伤残 = 21个月
- 五级伤残 = 18个月
- 六级伤残 = 16个月
- 七级伤残 = 13个月
- 八级伤残 = 11个月
- 九级伤残 = 9个月
- 十级伤残 = 7个月

## 4. 时效与程序核心限制
- **仲裁时效**：《劳动争议调解仲裁法》第27条，劳动争议申请仲裁的时效期间为**一年**。从当事人知道或者应当知道其权利被侵害之日起计算。劳动关系存续期间因拖欠劳动报酬发生争议的，劳动者申请仲裁不受一年仲裁时效期间的限制；但是，劳动关系终止的，应当自劳动关系终止之日起一年内提出。
- **加班费基数**：应当按照劳动合同约定的劳动者本人小时工资标准或实际月薪剔除非工作津贴后计算。工作日加班150%，休息日加班200%，法定节假日加班300%。
