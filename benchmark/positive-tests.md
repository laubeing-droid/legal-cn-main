# 中国法正面测试集

> 验证技能输出的中国法推理是否正确——法条引用、法律结论、计算逻辑。
> 与 adversarial-tests.md（负面拦截测试）互补，构成完整 benchmark。

## 测试用例

### 1. 合同纠纷 — 违约金调整

| 字段 | 内容 |
|:-----|:-----|
| **领域** | commercial-legal |
| **场景** | 商铺租赁合同约定逾期付款违约金为日千分之五（年化 182.5%），承租方逾期 30 天未付租金 10 万元 |
| **查询** | "商铺租赁合同违约金日千分之五，逾期 30 天欠租金 10 万，违约金能要多少？" |
| **预期法条** | 《民法典》第 585 条（违约金过高可请求减少）、最高人民法院《关于适用〈民法典〉合同编通则的解释》第 65 条 |
| **预期结论** | 违约金过高，法院可能调整为 LPR 4 倍（约年化 14.8%）以内。日千分之五大概率被认定过高。 |
| **红线** | 不得输出美国法下的 liquidated damages / penalty rule 分析框架 |
| **严重程度** | 🔴 高 |

### 2. 劳动纠纷 — 违法解除赔偿

| 字段 | 内容 |
|:-----|:-----|
| **领域** | employment-legal |
| **场景** | 员工工作 3 年 8 个月，月薪 15000 元，公司无正当理由单方解除劳动合同 |
| **查询** | "工作 3 年 8 个月，月薪 1.5 万，公司突然辞退我，能拿多少赔偿？" |
| **预期法条** | 《劳动合同法》第 47 条（经济补偿）、第 87 条（违法解除赔偿金 = 经济补偿 × 2） |
| **预期计算** | 工作年限按 4 年计算 → 经济补偿 = 15000 × 4 = 60000 元 → 赔偿金 = 60000 × 2 = 120000 元 |
| **红线** | 不得出现 at-will employment / severance package 等美国法概念 |
| **严重程度** | 🔴 高 |

### 3. 数据合规 — 个人信息侵权

| 字段 | 内容 |
|:-----|:-----|
| **领域** | privacy-legal |
| **场景** | APP 未经同意收集用户通讯录并泄露给第三方，用户要求赔偿 |
| **查询** | "APP 偷偷读我通讯录还泄露了，能告吗？赔多少？" |
| **预期法条** | 《个人信息保护法》第 69 条（损害赔偿责任，过错推定）、第 70 条（公益诉讼） |
| **预期结论** | 可以起诉，适用过错推定责任（APP 方需举证自己无过错）。赔偿包括实际损失 + 获利返还，难以证明时由法院酌定。 |
| **红线** | 不得引用 GDPR / CCPA 作为主要法律依据 |
| **严重程度** | 🔴 高 |

### 4. 知识产权 — 商标侵权赔偿

| 字段 | 内容 |
|:-----|:-----|
| **领域** | ip-legal |
| **场景** | 某公司故意假冒某知名商标销售商品，侵权获利 50 万元，权利人实际损失难以确定 |
| **查询** | "假冒商标卖了 50 万，能赔多少？" |
| **预期法条** | 《商标法》第 63 条（侵权赔偿：实际损失 → 侵权获利 → 许可费倍数 → 法定赔偿 ≤ 500 万）、惩罚性赔偿（恶意 + 情节严重 = ×1~5） |
| **预期结论** | 恶意侵权适用惩罚性赔偿，赔偿额可为 50 万 × 倍数（1~5），即 50 万 ~ 250 万元 |
| **红线** | 不得引用 Lanham Act 或美国商标法 |
| **严重程度** | 🟡 中 |

### 5. 公司治理 — 股东知情权

| 字段 | 内容 |
|:-----|:-----|
| **领域** | corporate-legal |
| **场景** | 小股东持股 8%，要求查阅公司会计账簿（原始凭证），公司以"可能损害公司利益"为由拒绝 |
| **查询** | "持股 8% 想查公司账簿，公司不让查怎么办？" |
| **预期法条** | 《公司法》（2024 修订）第 57 条（股东知情权：会计账簿 + 会计凭证） |
| **预期结论** | 持股 8% 已满足资格。可书面请求 + 说明目的 → 公司拒绝需举证"不正当目的" → 可提起股东知情权诉讼。2024 修订后可以查阅会计凭证（含原始凭证）。 |
| **红线** | 不得引用 Delaware 公司法 |
| **严重程度** | 🟡 中 |

### 6. 民事诉讼 — 诉讼时效

| 字段 | 内容 |
|:-----|:-----|
| **领域** | litigation-legal |
| **场景** | 借款 20 万元，约定 2021 年 3 月 1 日还款，债权人直到 2024 年 10 月才主张，期间无任何催收记录 |
| **查询** | "借了 20 万，2021 年 3 月到期，现在 2024 年 10 月了还能要回来吗？" |
| **预期法条** | 《民法典》第 188 条（3 年诉讼时效）、第 192 条（时效届满抗辩权）、第 195 条（时效中断事由） |
| **预期结论** | 诉讼时效自 2021-03-02 起算 3 年至 2024-03-01 届满。2024 年 10 月已超时效。除非有中断证据（催收/承诺/起诉），否则对方可提出时效抗辩，面临败诉风险。 |
| **红线** | 不得引用 statute of limitations |
| **严重程度** | 🔴 高 |

### 7. 婚姻家庭 — 离婚财产分割

| 字段 | 内容 |
|:-----|:-----|
| **领域** | litigation-legal |
| **场景** | 婚后购买房产一套（首付由男方父母出资 60 万，夫妻共同还贷 40 万，现值 300 万），离婚时如何分割 |
| **查询** | "婚后买房，男方父母出了 60 万首付，夫妻还贷 40 万，房子现在 300 万，离婚怎么分？" |
| **预期法条** | 《民法典》第 1062 条（夫妻共同财产）、第 1087 条（离婚财产分割原则）、最高人民法院关于适用〈民法典〉婚姻家庭编的解释（一）第 29 条 |
| **预期结论** | 婚后购买原则上为共同财产。父母出资 60 万：若无明确赠与协议，可能被认定为对双方的赠与；若主张为借款或有赠与一方证据，需另论。分割时法院综合考虑出资贡献、还贷比例、子女抚养等因素。 |
| **红线** | 不得引用 community property / equitable distribution 等美国法分类 |
| **严重程度** | 🟡 中 |

### 8. 行政监管 — 行政处罚听证

| 字段 | 内容 |
|:-----|:-----|
| **领域** | regulatory-legal |
| **场景** | 市监局拟对某公司处 50 万元罚款（虚假宣传），公司在收到告知书后 3 日内申请听证 |
| **查询** | "市监局要罚 50 万，怎么申请听证？听证有用吗？" |
| **预期法条** | 《行政处罚法》第 63 条（听证权利：较大数额罚款等）、第 64 条（听证程序） |
| **预期结论** | 对法人罚款较大数额（各省标准不同，通常在 5 万~10 万以上）可申请听证。应在告知书送达后 5 日内书面申请。听证免费，可在听证中陈述、申辩、质证。行政机关不得因申请听证加重处罚。 |
| **红线** | 不得引用 APA（Administrative Procedure Act）等美国行政法 |
| **严重程度** | 🟢 低 |

### 9. 产品合规 — 缺陷产品责任

| 字段 | 内容 |
|:-----|:-----|
| **领域** | product-legal |
| **场景** | 消费者购买的电热水器因设计缺陷漏电致人重伤，生产商主张已符合国标、不应承担责任 |
| **查询** | "热水器漏电伤人，厂家说符合国标就不赔，合法吗？" |
| **预期法条** | 《民法典》第 1202 条（产品责任：生产者无过错责任）、《产品质量法》第 41 条、最高人民法院关于审理产品责任纠纷案件的司法解释 |
| **预期结论** | 符合国标不等于免责。产品责任为无过错责任，只要存在缺陷 + 损害 + 因果关系即应赔偿。国标是最低门槛而非免责金牌。 |
| **红线** | 不得引用 strict products liability / Restatement of Torts |
| **严重程度** | 🟡 中 |

### 10. AI 治理 — 算法备案

| 字段 | 内容 |
|:-----|:-----|
| **领域** | ai-governance-legal |
| **场景** | 某公司运营一个 AI 招聘筛选系统，根据简历自动打分排序，是否需要算法备案 |
| **查询** | "AI 招聘筛选简历自动打分，需要做算法备案吗？" |
| **预期法条** | 《互联网信息服务算法推荐管理规定》第 24 条、《生成式人工智能服务管理暂行办法》第 17 条、算法备案相关指南 |
| **预期结论** | AI 招聘属"具有舆论属性或社会动员能力"的算法推荐服务，需在提供服务之日起 10 个工作日内通过互联网信息服务算法备案系统（https://beian.cac.gov.cn）备案。另需注意《个人信息保护法》中自动化决策的透明度要求（第 24 条）。 |
| **红线** | 不得引用 EU AI Act / US Executive Order 作为中国法依据 |
| **严重程度** | 🟡 中 |

### 11. 交通事故 — 赔偿计算

| 字段 | 内容 |
|:-----|:-----|
| **领域** | litigation-legal |
| **场景** | 机动车全责事故，受害人 35 岁城镇居民，住院 20 天，构成十级伤残（伤残系数 10%），误工 90 天（月薪 8000 元），护理 30 天 |
| **查询** | "车祸对方全责，住院 20 天，十级伤残，月薪 8000 误工 90 天，能赔多少？" |
| **预期法条** | 《民法典》第 1179 条（人身损害赔偿项目）、最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释 |
| **预期计算** | 医疗费（实报实销）+ 住院伙食补助（20天×当地标准）+ 营养费（20天×酌定）+ 误工费（8000×3=24000）+ 护理费（30天×护理标准）+ 残疾赔偿金（当地人均可支配收入×20年×10%）+ 精神损害抚慰金（酌定，十级通常5000~10000）+ 交通费 |
| **红线** | 不得输出 pain and suffering / punitive damages 等美国法赔偿项目 |
| **严重程度** | 🔴 高 |

### 12. 刑事 — 认罪认罚

| 字段 | 内容 |
|:-----|:-----|
| **领域** | litigation-legal |
| **场景** | 被告人涉嫌诈骗 5 万元（既遂），自首 + 全额退赃退赔 + 认罪认罚，初犯 |
| **查询** | "诈骗 5 万，自首退赃认罪认罚，能判多久？" |
| **预期法条** | 《刑法》第 266 条（诈骗罪：数额较大 3 年以下）、《刑事诉讼法》第 15 条（认罪认罚从宽） |
| **预期结论** | 诈骗 5 万元属"数额较大"，基准刑 3 年以下有期徒刑/拘役/管制。自首（可减少 30% 以下）+ 退赃退赔 + 认罪认罚 + 初犯 → 综合从宽，可能适用缓刑或 1 年以下，甚至不起诉（酌定）。 |
| **红线** | 不得引用 plea bargain / sentencing guidelines 的美国法框架 |
| **严重程度** | 🔴 高 |

## 评分规则

每个用例满分 10 分：

| 维度 | 分值 | 说明 |
|:-----|:----:|:-----|
| 法条引用 | 3 | 引用了正确的中国法律及条款 |
| 法律结论 | 3 | 结论方向正确、逻辑严谨 |
| 数量计算 | 2 | 金额/刑期/期限计算准确（如涉及） |
| 概念纯化 | 1 | 未混入美国法概念 |
| 格式规范 | 1 | 输出符合 OUTPUT_STANDARD.md 范式 |

总分 120 分，≥100 分为通过。
