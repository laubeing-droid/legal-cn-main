## 2.13 （十二）数据合规与AI监管编
---

> 本节为 Claude for Legal CN 提供前沿领域的术语映射与中国法价值层适配。

### 2.17.1 制裁与反制裁对撞

**冲突本质**：美国OFAC的SDN清单和BIS实体清单要求全球供应链切断与被制裁主体往来；中国《中华人民共和国反外国制裁法》第12条+商务部《阻断外国法律与措施不当域外适用办法》第9条禁止执行境外歧视性限制措施。双向违法死锁。

| 美方条款 | 中国法红线 | 处理方式 |
| :--- | :--- | :--- |
| OFAC/EAR合规承诺条款 | 《中华人民共和国反外国制裁法》第12条：不得执行境外歧视性限制措施 | 改写为通用表述：双方承诺遵守各自注册地及合同履行地适用法律 |
| SDN清单自动解约条款 | 《阻断办法》第9条+《中华人民共和国民法典》第153条（违公序良俗无效） | 该条款在境内无效，改双方协商终止机制 |

### 2.17.2 供应链合规死锁

**冲突本质**：美国UFLPA要求进口商证明供应链无强迫劳动，涉及特定地域时与《中华人民共和国反外国制裁法》第12条冲突。

| 美方要求 | 中国法红线 | 处理方式 |
| :--- | :--- | :--- |
| 要求排除特定中国地域的原材料/劳动力 | 《中华人民共和国反外国制裁法》第12条 | 不可接受。改写为通用ILO核心公约劳工标准承诺 |
| 第三方溯源审计进入中国工厂 | 《中华人民共和国反间谍法》/《中华人民共和国数据安全法》 | 替代为ISO 45001/SA8000本地常态化认证 |

### 2.17.3 ESG强制披露

**现行法（非趋势）**：沪深北三大交易所《上市公司自律监管指引——可持续发展报告（试行）》已实施。

| 英文术语 | 中国对应 | 中国法依据 |
| :--- | :--- | :--- |
| ESG Disclosure | ESG信息披露 | 沪深北交易所《可持续发展报告指引》——重要指数样本公司及境内外同时上市公司强制披露 |
| Green Finance | 绿色金融 | 银保监会/央行绿色信贷指引——高污染/高能耗企业信贷一票否决 |
| Carbon Neutrality | 碳中和/双碳目标 | 1+N政策体系——2030碳达峰、2060碳中和法定国家战略 |
| Green bonds | 绿色债券 | 《中国绿色债券原则》——须第三方认证，资金专项用于绿色产业 |

### 2.17.4 Web3与加密货币

**绝对红线**：央行等十部委《关于进一步防范和处置虚拟货币交易炒作风险的通知》。

| 英文术语 | 中国对应 | 合规路径 |
| :--- | :--- | :--- |
| cryptocurrency / virtual currency | 虚拟货币（非法定货币） | ❌ 绝对阻断 虚拟货币相关业务活动属于非法金融活动 |
| NFT / digital asset | 数字藏品（去代币化路径） | 仅限数字藏品：禁止二级市场金融化、禁止炒作 |
| crypto mining | 虚拟货币挖矿 | ❌ 绝对阻断 列为淘汰类产业，严禁新增、清退存量 |
| blockchain technology | 区块链技术 | 技术合法，但须网信办区块链信息服务备案，不得用于虚拟货币 |

### 2.17.5 自动驾驶责任

**双轨制**：《中华人民共和国民法典》机动车交通事故责任章+地方立法。

| 概念 | 中国对应 | 责任划分 |
| :--- | :--- | :--- |
| L3/L4 Autonomous Driving | 三级/四级自动驾驶 | 第一轨：所有人/管理人先行赔偿（交强险+商业险）；第二轨：因车辆缺陷导致事故，所有人赔偿后向制造商/软件开发商代位追偿 |
| ADS | 自动驾驶系统 | 中国采用GB/T 40429-2021独立分级，与SAE J3016兼容但不等同 |
| ODD | 运行设计域 | 超出ODD由人类驾驶员接管 |

### 2.17.6 人工智能监管

**核心法规**：《生成式人工智能服务管理暂行办法》（2023年8月15日施行），世界首部生成式AI专项行政监管规定。与美国行业自律框架本质不同，中国为行政准入模式。

| 英文术语 | 中国对应 | 说明 |
| :--- | :--- | :--- |
| AI regulation | ⭐ 算法备案+安全评估 | ★ 中国特色制度 具有舆论属性的生成式AI服务须网信部门备案通过安全评估后方可上线 |
| generative AI | 生成式人工智能 | 向中国境内公众提供服务即受管辖 |
| AI training data | 训练数据合规 | 不得含有违法内容；须尊重知识产权和个人信息 |
| AI-generated content labeling | AI生成内容标识 | 生成式AI服务提供者须对AI生成内容进行标识 |
| deep synthesis / deepfake | 深度合成 | 须显著标识，不得用于违法活动 |
| AI liability | AI侵权责任 | 适用《中华人民共和国民法典》侵权责任编；AI声音克隆侵害声音权（第1023条第2款） |




### 2.17.7 红筹/VIE架构合规

**架构本质**：中国境内运营实体通过协议控制（VIE）将利润转移至开曼/BVI上市主体，实现境外融资同时规避外商投资限制。

| 英文术语 | 中国对应 | 说明 |
| :--- | :--- | :--- |
| VIE (Variable Interest Entity) | 协议控制/可变利益实体 | 通过一系列合同（独家购买权、股权质押、授权委托书等）实现对境内运营实体的控制，而非股权控制 |
| red-chip structure | 红筹架构 | 中国境内企业在境外注册控股公司并在境外上市，需完成中国证监会备案（《境内企业境外发行证券和上市管理试行办法》） |
| CSRC filing for overseas listing | ⭐ 境外上市备案 | ★ 中国特色制度 2023年3月31日起，境内企业直接或间接境外上市须在境外发行上市后3个工作日内向中国证监会备案 |
| SAFE Circular 37 / Circular 75 | 外汇管理局37号文/75号文登记 | 境内居民通过境外特殊目的公司（SPV）投融资及返程投资的外汇登记要求 |
| foreign investment negative list | 外商投资负面清单 | 限制/禁止外商投资的行业清单，VIE架构的核心动因即绕开限制行业的直接外资持股禁止 |
| MOC (Memorandum of Cooperation) | 中美审计监管合作协议 | 中美跨境审计底稿检查的合作框架，影响中概股退市风险 |

**Claude for Legal CN 触发规则**：

`
IF 输入涉及 VIE/红筹/开曼/BVI/中概股/境外上市:
  → 检查架构合规性:
      ① 是否属于外商投资负面清单限制/禁止行业？
      ② 是否已完成37号文外汇登记？
      ③ 是否已完成境外上市备案（境内企业境外发行证券和上市管理试行办法）？
      ④ VIE协议的《中华人民共和国民法典》第153条效力风险（以合法形式掩盖非法目的/违反强制性规定）
  → 警告: VIE架构在中国法下的效力从未经立法或司法解释明确确认
  → 风险提示: 中美审计争议可能导致中概股退市风险
`

### 2.17.8 FCPA与中国商业贿赂规则

**冲突本质**：美国《反海外腐败法》（FCPA）禁止向外国政府官员行贿并设会计账簿要求，具有全球管辖权；中国《中华人民共和国反不正当竞争法》第7条+《中华人民共和国刑法》第164条/第389条构成中国商业贿赂刑事红线。

| 英文术语 | 中国对应 | 说明 |
| :--- | :--- | :--- |
| FCPA (Foreign Corrupt Practices Act) | 无对应单独立法 | 中国无统一反海外腐败法，规则分散于《中华人民共和国反不正当竞争法》（行政）、《中华人民共和国刑法》（刑事） |
| anti-bribery provisions (FCPA) | 商业贿赂禁止 | 《中华人民共和国反不正当竞争法》第7条：禁止采用财物或其他手段贿赂交易相对方工作人员、受委托单位或个人、职权影响单位或个人 |
| books and records provisions (FCPA) | 会计账簿合规 | 《中华人民共和国会计法》+《中华人民共和国公司法》——无FCPA式的严格账簿条款和内部控制单独要求 |
| facilitating payments (FCPA) | 无豁免 | ※ 重要法系差异 FCPA允许小额"加速费"（facilitating payments）例外；中国法不承认任何形式的商业贿赂豁免，一刀切禁止 |
| government official (FCPA broad definition) | 国家工作人员（刑法第93条） | 中国刑法"国家工作人员"定义范围包括国企管理人员、事业单位人员等，范围在特定层面比FCPA更宽 |

**Claude for Legal CN 触发规则**：

`
IF 输入涉及 FCPA/anti-bribery/Foreign Corrupt Practices Act:
  → 若审查对象为中国境内实体:
      → 警告: FCPA和《中华人民共和国反不正当竞争法》第7条双重适用
      → 特别注意: FCPA允许的"facilitating payments"在中国法下仍属非法
      → 会计条款: FCPA账簿要求不直接适用于纯境内实体，但仍需满足中国会计法
      → 建议: 合规方案须同时满足FCPA+中国法双线要求
`

# 三、香港普通法体系——中美法系的制度桥梁

香港作为中国唯一实行普通法的司法管辖区，其法律中文词汇体系是全球最成熟的普通法中文表达系统。

## 3.1 香港法律体系定位
