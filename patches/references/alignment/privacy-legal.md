# 数据合规领域 - 中美法律概念对齐
## 加载范围
- §2.13 数据合规与AI监管编 + §1.6.4 数据出境触发规则
---

中国建立了以"三法一条例"（《中华人民共和国网络安全法》《中华人民共和国数据安全法》《中华人民共和国个人信息保护法》《生成式人工智能服务管理暂行办法》）为核心的数据与AI监管体系。

| 英文术语 | 功能对应译法 | 对应类型 | 强度 | 对应法条 | 风险标签 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| personal data / PII | 个人信息 | 完全对应 | ★★★★★ | 《中华人民共和国个人信息保护法》第4条 | |
| sensitive personal data | 敏感个人信息 | 完全对应 | ★★★★★ | 《中华人民共和国个人信息保护法》第28条 | |
| data controller | 个人信息处理者 | 完全对应 | ★★★★★ | 《中华人民共和国个人信息保护法》第5条 | |
| data processor | 受托处理者 | 完全对应 | ★★★★★ | 《中华人民共和国个人信息保护法》第21条 | |
| **cross-border data transfer** | **数据出境** | 功能对应 | ★★★★ | 《中华人民共和国个人信息保护法》第38条、《中华人民共和国数据安全法》第31条 | ⚠ 高风险误译 三轨制：安全评估/标准合同/认证 |
| **data localization** | **数据本地化存储** | 完全对应 | ★★★★★ | 《中华人民共和国网络安全法》第37条 | 关键信息基础设施运营者须境内存储 |
| **important data / core data** | ⭐ 重要数据/核心数据 | ★ 中国特色制度 | ★★★★★ | 《中华人民共和国数据安全法》第21条 | ★ 中国特色制度 基于国家安全独创的数据分级分类 |
| ⭐ 数据出境安全评估 | ⭐ 数据出境安全评估 | ★ 中国特色制度 | ★★★★★ | 《中华人民共和国个人信息保护法》第38条 | ★ 中国特色制度 |
| ⭐ 算法备案 | ⭐ 算法备案 | ★ 中国特色制度 | ★★★★★ | 《互联网信息服务算法推荐管理规定》/《生成式人工智能服务管理暂行办法》 | ★ 中国特色制度 生成式AI服务上线前须完成算法备案与安全评估（网信部门前置审批） |

**涉外律师实务话术**：
- "中国数据出境实行三轨制：安全评估、标准合同、个人信息保护认证。处理100万人以上个人信息或累计出境10万人/1万人敏感信息的，须通过国家网信部门安全评估。"
- "算法备案是中国生成式AI服务的法定市场准入前置条件，不同于美国自愿性质的AI伦理框架。"
