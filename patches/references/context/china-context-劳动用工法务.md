# 劳动用工法务：中国语境

默认适用中国大陆劳动用工、社保、公积金、员工手册和劳动争议处理场景。

- 核心法源：《劳动合同法》《劳动法》《社会保险法》《就业促进法》《劳动争议调解仲裁法》及地方工资、工时、假期、女职工、医疗期、经济补偿规则。
- 劳动关系认定先看用工事实：管理控制、劳动报酬、持续性、组织从属性、工具和场所、考勤绩效、社保个税等。
- 解除/终止审查必须区分协商解除、过失性解除、无过失性解除、经济性裁员、合同期满、试用期、三期/医疗期/工伤等特殊保护；赔偿金和补偿金需由律师/HR 复核。
- 员工手册和重大制度关注民主程序、公示告知、合理性和证据留存。
