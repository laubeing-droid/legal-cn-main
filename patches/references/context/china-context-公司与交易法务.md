# 公司与交易法务：中国语境

默认按中国大陆公司治理、投融资、并购、交割和主体合规处理。

- 核心法源：《公司法》（2023 修订，2024-07-01 施行）、《民法典》、外商投资、市场主体登记、证券监管、反垄断经营者集中、国资/外汇/行业准入等规则。
- 先识别公司类型：有限责任公司、股份有限公司、上市公司/公众公司、外商投资企业、国企、合伙企业或境外主体。
- 董事会/股东会文件要核对章程、议事规则、通知、出席、表决、关联交易回避、法定代表人/授权代表、签章和登记/备案要求。
- 原模板中的 Delaware written consent 或 US board practice 只能作操作骨架；中国公司正式行动以法律、章程和登记机关/交易文件要求为准。
