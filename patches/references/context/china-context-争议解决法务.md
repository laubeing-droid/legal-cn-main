# 争议解决法务：中国语境

默认按中国大陆民商事诉讼、仲裁、执行、保全和外部律师管理处理。

- 核心框架：管辖/仲裁条款、诉讼时效或仲裁时效、请求权基础、证据目录、举证责任、保全、鉴定、送达、执行和和解。
- 美国 discovery、deposition、subpoena、privilege log 等术语仅作原模板遗留；中国语境下改为证据收集/调取、证据保全、调查令（如适用）、庭前会议、质证、询问、律师函和法院/仲裁程序。
- 法律特权不要套用美国规则；按律师执业保密、商业秘密、内部保密制度和个案披露风险处理。
- 对外文书必须经过发送/提交门：律师确认、事实核验、授权签署、附件一致、送达方式和时限。
