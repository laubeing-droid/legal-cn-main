# 数据合规与个人信息保护：中国语境

默认适用中国大陆个人信息和数据合规框架。

- 核心法源：《个人信息保护法》《数据安全法》《网络安全法》及 CAC、MIIT、公安、网安标委等规则/标准。
- PIA 在中国语境下通常对应“个人信息保护影响评估（PIPIA）”，重点覆盖处理目的、合法性基础、最小必要、告知同意/单独同意、敏感个人信息、未成年人、自动化决策、委托处理、共同处理、向第三方提供、公开披露、保存期限、安全措施和个人权利响应。
- 数据出境先判断是否涉及个人信息、敏感个人信息、重要数据、关键信息基础设施运营者；再判断安全评估、标准合同、认证、豁免或自贸区负面清单路径。
- DSAR 改为个人信息主体权利请求，包括查阅、复制、更正、补充、删除、撤回同意、注销账号、解释说明自动化决策等。
