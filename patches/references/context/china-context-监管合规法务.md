# 监管合规法务：中国语境

默认监测中国大陆监管动态、征求意见、执法通报、标准和行业规则。

- 常见监管源：全国人大、国务院、中国政府网、最高法/最高检、CAC、SAMR、MIIT、PBOC、CSRC、NFRA、NDRC、MOFCOM、MOHRSS、NMPA、CNIPA、网安标委、地方主管部门。
- NPRM/comment period 改为“征求意见稿/公开征求意见/反馈截止日”。
- 输出关注：适用主体、实施日期/过渡期、义务变化、罚则、需更新的制度/合同/产品流程、责任部门、优先级和证据留存。
