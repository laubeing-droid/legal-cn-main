# 商事合同法务：中国语境

默认按中国大陆商事合同和企业法务场景处理。

- 适用主线：《民法典》合同编、担保制度、格式条款规则、电子签名、商业秘密、反不正当竞争、广告/消费者保护、个人信息保护、数据安全和网络安全。
- 合同角色：先判断我方是供应商/服务商、采购方/客户、平台方、渠道方、代理/经销方或合作方。
- 争议解决：重点审查适用法律、法院管辖、仲裁机构（CIETAC/北仲/上仲/深国仲等）、仲裁地、语言、保全、送达和执行。
- 数据与技术：DPA、数据出境、软件/SaaS、API、云服务、开源、AI 输出、训练数据、日志留存和安全事件条款需单独列为风险项。
- 对外版本默认不带内部审查标记；内部 review memo 可使用“保密 / 内部法律分析”。
