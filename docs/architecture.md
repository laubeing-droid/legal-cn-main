# 架构文档

> 最后更新：2026-05-25

---

## 核心问题

> 中国法律 AI 的开源生态里，所有人都在做同一件事：**把 anthropic 的美国法改成中文名。**
>
> 但 99% 的人只做到了表层（翻译 UI、替换法条名）。**真正的问题不是"名字翻译了没有"，而是"AI 会不会在回答中国法律问题时，脑子里还是美国法的逻辑"。**
>
> 本仓库就是为了解决这个问题。

---

## 六层适配架构

每一层解决一个"美国法→中国法"的实质问题：

```
┌──────────────────────────────────────────────────┐
│ ⑥ 上游监控层（本仓库独创）                        │
│                                                    │
│    4 路 diff-tool 各自跟踪一个上游                 │
│    解决的问题：上游更新了怎么办？                  │
│    答案：→ 通知你，你自己决定，不自动覆盖          │
├──────────────────────────────────────────────────┤
│ ⑤ 护栏层（自研，本仓库独有）                      │
│                                                    │
│    阻断清单 / 元规则 / 香港桥梁 / 核心原则          │
│    解决的问题：AI 会不会混入美国法概念？           │
│    答案：→ blocking-list.md 列出 22 个禁止概念     │
│          → meta-rules.md 约束回答框架              │
│          → hk-bridge.md 处理香港普通法问题         │
├──────────────────────────────────────────────────┤
│ ④ PRC-US 概念对齐层（自研，本仓库独有）           │
│                                                    │
│    12 个领域 × 一对一概念映射                       │
│    解决的问题：AI 认识中国法概念吗？               │
│    答案：→ "美国 discovery = 中国举证/调查令"      │
│          → "美国 deposition = 中国没有这回事"      │
│          → "美国 consideration = 中国对价"         │
├──────────────────────────────────────────────────┤
│ ③ 法条层（来自 Daknniel-0881/qulv 知识库）                      │

│    22 部中国法律官方 PDF 全文                       │
│    解决的问题：之前只有摘要，没有原文               │

├──────────────────────────────────────────────────┤

├──────────────────────────────────────────────────┤
│ ① 部署层（本仓库重写）                            │
│                                                    │
│    install/update 从本仓库直接部署                  │

│    答案：→ 你改了什么，用户就装什么                │
│          → 上游变"参考窗口"                        │
└──────────────────────────────────────────────────┘
```

## 部署流程

```powershell
install.ps1
├── 12 个法律领域（从本仓库 skills/ 复制）
├── solo-law-firm（8 科室嵌套结构）
├── 克隆 MCP Hub
└── 验证 14 个入口
```

## 更新检测

```powershell
diff-tool-zhou.ps1 [-Diff] [-Update]
├── 拉取 zhou210712（TEMP 目录，保留缓存）
├── 比对 174 个文件（哈希 = 快照 vs 上游）
├── 输出颜色标记
└── -Diff 显示行级差异
```
